## Goals

By the end of this week, you should be able to:

- Open a local project from GitHub
- Run JavaScript in the browser
- Use the browser Console
- Make a small change to a JavaScript file
- Commit and push work to your GitHub repository

---

## Day 1 (Tuesday)

Today is focused on setup and orientation.

You will:

- Fork and clone the course repository
- Open a local HTML file in the browser
- Open the Developer Tools Console
- Confirm that JavaScript is running

There is **no submission** required today.

---

## What to do (Day 1)

1. Open `index.html` in your browser.
2. Open the browser Developer Tools.
3. Click on the **Console** tab.
4. Look for a message that says: `Week 1 Loaded`

If you see this message, JavaScript is running correctly.

---

## Day 2 (Thursday)

Today is focused on making small changes and submitting work.

You will:

- Update your project with changes from the instructor repository
- Modify a JavaScript file
- Use the Console to verify behavior
- Commit and push your work during class

---

## What to do (Day 2)

1. Update your local project with the latest changes from the instructor repository.
2. Open `script.js`.
3. Modify the console message as instructed in class.
4. Reload the page and confirm the updated message appears in the Console.
5. Commit and push your changes to your GitHub repository before leaving class.
6. Ensure it's published to GitHub Pages and submit the link via HuskyCT

---

## Notes
